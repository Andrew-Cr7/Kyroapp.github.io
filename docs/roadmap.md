# Kyro Roadmap

## Purpose

This document defines Kyro's strategic priorities.

The purpose is to ensure that all product, marketing and technical decisions support the current stage of the business.

When making decisions, always prioritise work that moves Kyro closer to launch, growth and revenue.

---

# Current Stage

Status:

Pre-Launch

Current Focus:

Building supply, demand and awareness before public launch.

Kyro is currently validating the marketplace and preparing for launch in London.

---

# North Star Goal

Become the easiest way for people to access gyms anywhere in the world.

A user should be able to:

1. Arrive in a city
2. Open Kyro
3. Find a gym
4. Purchase access
5. Start training

Within minutes.

# Features We Will Not Prioritise

The following are intentionally deprioritised:

* Workout tracking
* Nutrition tracking
* Fitness coaching
* Social media feeds
* Community forums
* Challenges
* Leaderboards
* Wearable integrations

These features do not currently solve the core problem.

---

# Decision Framework

Before starting any new project, ask:

1. Does this help acquire gyms?
2. Does this help acquire users?
3. Does this improve conversion?
4. Does this improve retention?
5. Does this generate revenue?

If the answer is no to all five, the work should probably not be prioritised.
