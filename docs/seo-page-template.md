# Kyro SEO Page Template

## Purpose

This document defines the structure for all Kyro SEO landing pages.

The goal is to create pages that:

* Rank in search engines
* Provide genuine value
* Convert visitors into waitlist signups
* Establish trust
* Support future pass purchases

Every location page should follow this framework unless there is a strong reason not to.

---

# Primary Goal

The primary goal of every SEO page is:

Convert organic visitors into waitlist signups.

Secondary goals:

* Increase brand awareness
* Improve search visibility
* Build trust
* Educate users about Kyro

---

# Target Search Intent

Users arriving on these pages are usually searching for:

* Gym day pass [city]
* Gym pass [city]
* Tourist gym [city]
* Visitor gym [city]
* Best gyms in [city]
* Day pass gym near me
* Gym for travellers [city]

The page should directly solve these problems.

---

# URL Structure

Format:

```text
/[city]-gym-day-passes
```

Examples:

```text
/london-gym-day-passes
/tokyo-gym-day-passes
/seoul-gym-day-passes
/bangkok-gym-day-passes
/singapore-gym-day-passes
```

Use lowercase only.

Use hyphens.

Avoid unnecessary words.

---
